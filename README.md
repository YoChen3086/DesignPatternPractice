- ### Chapter4:
    ### 升學求職兩不誤 --> 開放封閉原則(The Open-Closed Principle)
    ###### 軟體實體(類別、模組、函數等等)應該可以擴展，但是不可修改。

- ### Chapter5:
    ### 會修電腦不會修收音機? --> 依賴倒轉原則
    ###### A. 高層模組不應該依賴低層模組。兩個都應該依賴抽象。
    ###### B. 抽象不應該依賴細節。細節應該依賴抽象。
    ### Liskov替換原則
    ###### 子類型必須能夠替換掉他們的父類型。
